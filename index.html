[ SYSTEM LOGIC ]

数字をタップ！
押されていない数字が時計回りに回転
Tap a number!
The other 8 number rotate clockwise.

[ MISSION / SET UP ]

赤いボタンで０～１１を選択
Use Red Buttons to select 0-11.

0 ＋ [SET]：目指す形（＝HOME）を確認
0 + [SET]: View the Goal State (=HOME).

１～１１ ＋ [SET]：その手数の問題を出題
1-11 + [SET]: Generate N-step puzzle.

[ COMMANDS ]

INPUT：任意入力モード
（重複時は強制リセット）
INPUT: Manual mode
(Duplicates trigger a reset).

左から順番にタップしてください。
Tap the green numbers at the bottom
in order from left to right.
