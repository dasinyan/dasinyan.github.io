Tap a number!

[ MISSION / SET UP ]

赤いボタンで０～１１を選択
Use Red Buttons to select 0-11.

0 ＋ [SET]：目指す形（＝HOME）を確認
0 + [SET]: View the Goal State (=HOME).

１～１１ ＋ [SET]：その手数の問題を出題
1-11 + [SET]: Generate N-step puzzle.

[ COMMANDS ]

INPUT：任意入力モード
（重複時は強制リセット）
INPUT: Manual mode
(Duplicates trigger a reset).

[ ANSWER EXAMPLE ]

画面下部の緑色の数字は、
ＨＯＭＥでの数字の位置を表しています
例えば「1」と表示されている場合は、
現在どの数字が表示されているかに関係なく、
左上の数字をタップしてください
The green solution numbers at the bottom represent the target position (HOME layout). For example, if it says "1", tap the top-left tile, no matter what number is currently there!
